# Eleventy-Parcel Starter

Use this to start a static site with Eleventy and Parcel.

## Things to update:

- package file
- \_data folder
- css/01-config
- delete the pages you don't need

### Scripts

```
# Install dependencies
yarn install

# Run
yarn start

# Build for production
yarn build

# Generate SVG sprite
yarn svg
```
